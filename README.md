This is the final project of the subject Web Technologies 2: HTML, CSS and JavaScript (HF3M35/029) of the HND Web Development@EdinburghCollege.

Regarding the use of SASS SCSS I finally opted for using CSS. As the main features of SCSS have been added to CSS in the last 2 years. Being variables one of them.

JS had been used for the creation of several features such as the widely known technique to render databases (array of objects), edit how an array of videos is going to behave or adding a nav burger/cart menu, and a toast when clicking on add to cart.

AI has been used to create a skeleton (plus the OrganicMe template, which a similar color palette can be seen in this website too), add comments overall and assistance to find documentation and answer doubts.
Claude Code had been used to get most of the assets using the Pexels, PixaBay and Unsplash APIs.

The testing have been done with the following webpages and techniques

1. HTML Validation --> [validator.w3.org](https://validator.w3.org).
2. CSS Validation --> [jigsaw.w3.org/css-validator](https://jigsaw.w3.org/css-validator)
3. Device Testing --> Chromium inspector for iPhone 17, 17 Pro Max and for Samsung Galaxy S25 and A56.
4. Browser Testing --> Browsers used: Chromium, Firefox, Safari
5. Functionality Testing --> Template table, with a few functionalities per page.
